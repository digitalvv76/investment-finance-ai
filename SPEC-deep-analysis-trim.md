# SPEC — 深度分析精简 + 组合映射修正（deep_lane 输出过长）

> 交接：**V1 → V2**（deep_lane = main 流水线 = V2 地盘，§6）。诊断+格式契约由 V1 出，实现归 V2/main。
> 日期：2026-07-11 · 信道：origin/v1-stable。用户已逐轮定稿本格式。

## 1. 意图
有实时行情时的深度分析走 `ANALYSIS_PROMPT` 4 步全展开、`max_tokens=1500`，输出偏长（约 250-350+ 字）。用户要求**收紧到 ~250-300 字，深度集中在②传导+③组合映射**，并修正③（原被泛化成板块视角，丢了"映射到用户实际持仓"这层核心价值）。

## 2. 现状（main 真实代码，已 `git show origin/main` 核对）
- `ANALYSIS_PROMPT`：4 步 = 事件定性 / 传导路径 / 组合映射 / 置信度；`self._max_tokens=1500`（可被 `settings.yaml` deep.max_tokens 覆盖）。
- `NO_DATA_PROMPT`：无行情时的 2 段 flash note（100-180 词），已带 anti-fabrication（`SPEC-deep-analysis-stale-data.md` 的硬门禁：无行情禁价位/百分比/买卖建议）。

## 3. 定稿格式规则
| 段 | 规则 |
|----|------|
| ① 事件定性 | **精炼 1-2 句**，不铺陈"宏观/行业/个股"分类 |
| ② 传导路径 | **只留直接链**——去掉间接传导、去掉加密货币旁枝；点出市场层面直接受益股 |
| ③ **组合映射** | **必须映射到用户实际持仓 ∪ 关注股**；②点名的主受益股若不在用户仓里，③ 改指**同一受益链条中用户在跟踪的票**；单一行动场景；**方向词（偏多/偏空）仅当有实时行情时才给**；**禁具体价位与买卖操作**；可含 1 句反向风险 |
| ④ 置信度 | 高/中/低 + 缺什么，**1 行** |
| 全文 | **~250-300 字**，深度集中在 ②③ |

## 4. 参考输出（定稿样例，回归锚点 = news_id=3787 防务军费）
```
【深度分析】国防官员请求国会追加超 670 亿美元补充军费

① 事件定性
政府支出催化。国防部向国会追加申请逾 670 亿美元补充军费，属行业级采购利好；
资金若获批将直接转为装备订单，利好确定性高于一般政策表态。

② 传导路径
资金主要投向导弹、防空、战机、舰艇等主战装备采购，直接放大主力防务承包商的
在手订单(backlog)与营收预期。市场层面直接受益方为五大主承包商 LMT、RTX、NOC、GD、BA。

③ 组合映射
你未持有上述五大主承包商，但关注列表中的国防纯玩家与本轮用途高度重合：舰艇采购
直接利好造船龙头 HII，无人机需求利好 KTOS、AVAV，核/舰艇动力部件利好 BWXT，
PLTR 受益国防 AI 采购。方向偏多，其中 HII(舰艇)与拨款用途最直接。催化确认=国会
审批与拨款落地；反向风险=拨款被削减或延期。

④ 置信度
中——拨款仍需国会审批，最终规模与时点未定。
```
（约 285 字。此样例中 LMT/RTX/NOC/GD/BA 均不在用户关注列表；HII/KTOS/AVAV/BWXT/PLTR 在 → ③ 正确改指后者。）

## 5. 实现要点（V2）
- 改 `ANALYSIS_PROMPT` 4 步措辞按 §3；每步给明确长度/内容约束，并在 prompt 里写死"全文约 250-300 字，深度放②③，①④从简"。
- ③ 组合映射依赖**用户持仓 + 关注股注入到 prompt 上下文**（现 prompt 措辞 "Given the holdings and investment rules in the knowledge base" 暗示已注入 `extra_context`/knowledge base）——**请确认真的注入了 portfolio ∪ watchlist**，否则③无从映射，会退回泛泛板块（正是要修的病）。
- **方向词闸**：③ 的"偏多/偏空"**仅当本条抓到实时行情时**输出；无行情走 `NO_DATA_PROMPT` 时不得出现方向词。
- `max_tokens` 可下调作硬 backstop（如 1500→900）防跑飞，但**真正靠 prompt 字数预算**，别只靠截断（会断句）。
- `NO_DATA_PROMPT`：建议同样采用 ①②③④ 小标题结构以求一致，但**③不给方向词、全程无任何数字**（维持既有 anti-fabrication）。
- ⚠️ **不得回退 `SPEC-deep-analysis-stale-data.md` 的硬门禁**：价位/涨跌幅只能用真实行情串里的精确值，无则纯定性。本次是"缩短+组合映射"，防编数约束原样保留。

## 6. 验收
- 定性 ≤2 句；传导路径正文**不含**"加密货币""间接"类旁枝；③ 引用的股票 ∈ 用户持仓∪关注股（或明确"主受益股不在仓"并改指跟踪票）；全篇**无具体价位、无买卖操作指令**。
- 有实时行情 → ③ 可含方向词；无实时行情 → ③ 不含方向词且无任何数字。
- 全文 ~250-300 字。
- 具名回归：喂 news_id=3787 应产出贴近 §4 样例的结构与长度。
