## 第一步：相关性初筛（避免噪音推送）
首先判断该新闻是否与美股市场投资有直接或间接的显著关联。若属于以下任意一种**无关类型**，直接返回一个精简的过滤JSON，无需进行后续催化剂识别：
- 纯政治、选举、外交访问等，且不涉及具体经济政策、政府支出、行业补贴或特定上市公司。
- 仅影响非美市场（如A股、港股、日股）的本地政策或事件，且无在美上市标的或全球产业链映射（例如：中国内地房地产调控、不涉中概股的港股交易规则变动）。
- 纯体育、娱乐八卦、社会新闻、自然灾害（除非明确指向再保险、能源等美股板块的供给冲击）。
- 重复旧闻、无实质内容的财经评论、纯粹的加密货币行情波动且不涉及美股上市公司。
- 明显只对非权益类资产（如美国国债、外汇、大宗商品）产生影响，但无法传导至权益市场的孤立消息。

**⚠️ 反过滤例外（务必遵守，防止漏判高价值事件）**：以下情况**绝不可**因"标题未点名具体上市公司"而过滤——它们属于必须深挖的强催化，你应主动识别受益股/联动板块：
- 美国政府/联邦机构（能源部/国防部/商务部/DARPA 等）的**行业级补贴、巨额基金、国家级采购或储备计划**（如"能源部10亿关键矿产基金""CHIPS 520亿补贴""800亿核电投资"）。金额大 + 行业级 = 高优先，即使无单一纯玩家也要铺开受益面（关键矿产→MP/UUUU/ALB，核电→CCJ/LEU/OKLO，芯片→设备材料链）。**广度不是降级理由，是深挖理由。**
- 唯一可压低强度的政府案例：金额小（数亿级）**且**无任何上市受益标的（如小型煤电拨款）。
- **重要宏观数据发布**（CPI/PPI/PCE/非农/失业率/GDP/美联储利率决议/ISM PMI/零售销售）：这些数据**直接驱动利率预期和增长前景**，并通过估值模型、板块轮动、风险偏好三条路径**系统性传导至全部美股**。即使报道本身未点名具体公司，也必须识别为相关事件，进入第二步催化剂识别（通常归入第3类"领袖级人物终局预言"等效 — 宏观数据是市场的定价锚）。**绝不可**因"未点名个股"或"只影响国债"而过滤。

**无关新闻返回格式**：直接返回如下 JSON（仅此一种情况使用）：

## 第二步：财富效应催化剂识别（仅相关新闻执行）
若新闻通过初筛，立即依据以下五大催化剂类型进行判定（可多选）：
1. **政府/国家资本入股或资助**：美国联邦政府、国防部、能源部、DARPA 等机构直接入股、巨额合同、国家级贷款或补贴。关键词如"入股""持股""bailout""CHIPS Act""DARPA contract""DOE funding"等。
2. **AI 超级巨头绑定**：英伟达（NVIDIA）、微软、OpenAI 等直接股权投资、13F 持仓披露、战略合作、成为其 GTC/Ignite 等大会提及的解决方案伙伴。关键词如"NVIDIA invests""stake""partnership""named by Jensen""13F"等。
3. **领袖级人物的终局预言**：黄仁勋、马斯克、Sam Altman、Cathie Wood 等明确给出极端目标价或"下一个万亿"等表述。关键词如"trillion""next big thing""Ark Invest price target"等。
4. **硬核里程碑式批准/突破**：FDA 药物/器械批准、FAA 认证、首次量子优越性实证、聚变净能量增益等，使技术从研发变为确定商业资产。关键词如"FDA approval""FAA certification""quantum supremacy""breakthrough"等。
5. **极端空头挤压/模因引爆**：高做空比例叠加散户抱团或知名多头逼空声明。关键词如"short squeeze""WallStreetBets""memes""Roaring Kitty"等。
   ⚠️ **本类专属门槛（务必执行）**：只有**已经发生的真实挤压**才算强催化——即新闻描述股价已启动轧空/已暴涨/散户资金已实际涌入/逼空已在进行（如"today surged 40% on short squeeze""shares halted after retail buying"）。
   反之，**纯预测/观察/清单式提示**（如"或再现模因行情""值得关注这几只模因股""具备逼空潜力""watch these meme stocks""could squeeze"）→ **不作为触发事件**，返回 `is_event=false, notable=true`（若确有具体标的），**不上手机**。区分关键：**已发生的动作 vs 尚未发生的可能性**。

如果新闻相关但**不触发任何催化剂**，返回：
{"is_event": false, "notable": false, "ticker_hint": [], "reason": "no catalyst triggered"}

**在这条"无催化剂"返回里，额外判定 `notable`（关注股安全网用）**：
- `notable = true` 当且仅当：新闻涉及某上市公司**实质性、可能驱动股价的具体动作**——如分析师重大评级/目标价调整、板块异动传导到个股、重大合同/大客户、显著财报意外、明显放量涨跌、具体产品/运营里程碑。**宁缺毋滥，拿不准判 false。**
- `notable = false`：顺带提及、旧闻复述、泛泛观点/估值随笔、纯股价描述无催化剂、纯宏观无个股映射。
- `ticker_hint`：**读全文后**准确提取美股代码数组（Teva→["TEVA"] 而非 ARM；El Nino→[]）。宁缺毋滥、禁止臆测子串。notable=true 时必须给出关联的关注标的代码。

## 第三步：强度研判与板块标注（仅触发催化剂时）
当匹配到至少一类催化剂后，按以下要求生成完整信息：
- `intensity`：1-5 星，度量**预期价格波动的剧烈程度（不分涨跌方向）**。5=板块级剧烈波动（暴涨**或**避险抛售/暴跌）；4=个股大幅波动（暴涨**或**暴跌）；3=明显异动；2=温和变动；1=短期小幅波动。**强度只量"多剧烈"，涨/跌由 `direction` 单独判定——利空暴跌事件同样可以是 intensity=5。**

  **强度考量因素（按优先级排序）**：
  ① **时效性（权重最高 ⚠️）**：事件是**何时发生/公布的**？今天/昨天新发生 → 顶格；本周内 → 正常；数周/数月前已发生、现在只是深度报道/回顾梳理 → **大幅降级**（即使涉及政府入股等强催化类型，回顾性报道的强度也不应超过 ★2）。**关键区分：事件发生时间 ≠ 报道发表时间。WSJ 今天发表的深度调查，如果描述的是去年的事 → 事件时间是"去年"，强度上限 ★2。**
  ② 事件独家性：首发独家 vs 多家同时报道 vs 此前已被广泛报道
  ③ 市场预期充分度：完全出乎意料 vs 部分预期 vs 充分定价
  ④ 标的市值：小市值弹性更强（加分），大市值蓝筹弹性小（压一档）
  ⑤ 是否多催化剂共振
  ⑥ 信源确定性

- `timeliness`：事件时效性评级（新字段，用于审计追溯）：
  `immediate` — 今天/昨天新发生或新公布，市场尚未定价
  `recent` — 本周内发生，大部分市场参与者已知但细节仍在消化
  `retrospective_new` — 数周/数月前的事件，但报道**首次披露此前未知的关键细节**（需指出具体是什么新信息）
  `retrospective` — 数月/数年前的事件，市场早已充分定价，仅深度报道重新梳理旧事
  ⚠️ `retrospective` 类事件 intensity 上限 ★2（不管催化剂类型多强），`retrospective_new` 上限 ★3。
- `direction`：事件对**主要受影响标的**的方向——`up`（利好/追多）/ `down`（利空/做空规避）/ `neutral`。**与强度解耦**。
- `confirmed`：信源确定性布尔值。**官方公告 / 已经实际发生**（如已宣布、已暴跌、已获批、已成交）= `true`；`reportedly` / 据传 / 传闻 / 未证实 / 匿名消息源 / "或将" = `false`。⚠️ **利空（direction=down）事件只有 `confirmed=true` 才会上手机；未确认的利空一律静音 TG**——所以对**确凿已发生**的利空务必给 `true`，对**传闻性**利空务必给 `false`。拿不准倾向 `false`。
  ⚠️ `confirmed` 只管信源可靠性，**不管时效性**（时效性由 `timeliness` 字段独立评估）。即使事件确凿已发生（confirmed=true），如果发生在数周/数月前（timeliness=retrospective），强度仍应大幅降级。
- `sector_tags`：用 1~3 个主题标签概括最可能受影响的板块，如 "Quantum Computing""AI Foundry""Biotech""EV""Aerospace & Defense" 等，必须采用英文且为常见行业/主题名称。
- `headline_signal`：**用中文**写一句话核心交易逻辑，点出财富效应从何而来。
- `ticker_hint`：从新闻中提取的可能关联股票代码数组（美股优先），无则空数组。
- `risk_snapshot`：**用中文**写该事件最快落空或证伪的关键点，点明最大风险。

**完整事件返回格式**：
{
  "is_event": true,
  "event_types": [1, 3],
  "intensity": 5,
  "direction": "up",
  "timeliness": "immediate",
  "confirmed": true,
  "sector_tags": ["AI", "Semiconductors"],
  "headline_signal": "英伟达参股SoundHound，市场或将其视为AI语音入口加冕。",
  "ticker_hint": ["SOUN"],
  "risk_snapshot": "注意投资仅是财务性质，不包含技术排他协议。"
}

## 评级校准范例（少样本，务必对齐以下判定原则）
以下真实案例已由人工校准，用于校正你的强度判定与受益面识别。**核心原则：**
1. **政府资本一律大利好**：政府大额补贴、政府/机构入股、国家级资助都是强催化（默认 ★★★★ 起），不论最终成败——**看短期财富效应，不看长期回报**（如危机救助当日就是大利好，即使多年后可能亏损也不降级）。
2. **广度不降级**：大额政府计划/行业级补贴即使无单一纯玩家也是高优先，正解是深挖受益面、铺开联动股，绝不因分散而降级。
3. **小市值弹性加分**：同等事件下，受益标的市值越小、越纯，爆发力越强，强度上调（小市值中小盘 → 更易 ★★★★★；大市值蓝筹弹性小 → 压一档）。
4. **负面/零和同样是强催化**，触发做空或规避。**利空按自身剧烈程度评级**（`direction=down`），不再硬塞进利好标尺顶格。
5. **纯非美市场事件（A股/港股独有、无美股标的或全球产业链映射）→ 不推**（is_event=false）。
6. **信源置信度打折**：`reportedly` / 未证实传闻，尤其针对**万亿市值巨头**（单条合规传闻难撼动巨头板块级异动）→ 强度打折（通常压到 ★3）并置 `confirmed=false`；官方公告 / 已实际发生事件不打折（`confirmed=true`）。
7. **时效性是第一权重 ⏱️**：先判断事件发生时间，再评分。**今天发生的政府入股 = ★5；深度报道 11 个月前政府入股的旧闻 = is_event=false 或 ★2**。强度考量中，时效性权重 > 催化剂类型。`timeliness=retrospective` 的事件无论催化剂多强，上限 ★2。

- **国家入股（大利好 ★★★★★）**：「商务部将89亿芯片补贴转为英特尔9.9%股权」→ intensity=5, event_types=[1], direction=up, ticker_hint=["INTC"], 联动=["TSM","GFS","AMAT","LRCX"], sector=["SMH","SOXX"]。理由：政府大额补贴+入股=大利好，芯片龙头。
- **政府入股资助行业+小市值加分（★★★★★）**：「商务部向9家量子公司拨款20亿并取少数股权，IBM独获10亿」→ intensity=5, event_types=[1], direction=up, ticker_hint=["IBM","GFS","QBTS","RGTI","IONQ"], sector=["QTUM"]。理由：政府以入股方式资助整个行业=大利好，且量子多为小市值→弹性大加分。
- **危机救助也是大利好，看短期不看长期（★★★★★）**：「财政部TARP向通用汽车注资495亿获61%股权」→ intensity=5, event_types=[1], direction=up, ticker_hint=["GM"], 联动=["F"]。理由：政府巨额注资=当日短期大利好；**即使最终售股亏损（长期），也不因长期结果降级**——评级看事件当下的财富效应。
- **行业级补贴，广度不降级（★★★★）**：「CHIPS法案520亿补贴落地，台积电/三星/美光/格芯等12家分配」→ intensity=4, event_types=[1], direction=up, ticker_hint=["TSM","MU","GFS","TXN","INTC"], 联动=["AMAT","LRCX","KLAC","ASML"], sector=["SMH","SOXX"]。理由：受益面广不等于弱，须深挖设备/材料联动，**不因分散降级**；大盘蓝筹为主故 ★★★★ 非 ★★★★★。
- **负面/零和是强催化（★★★★ down）**：「英伟达发布首款PC处理器N1X，英特尔/AMD/高通集体暴跌」→ intensity=4, event_types=[], direction=down, confirmed=true, ticker_hint=["INTC","AMD","QCOM"], 龙头=["NVDA"], sector=["SMH","SOXX"]。理由：对三家竞对是−（做空/规避对象放 ticker_hint），已实际暴跌=confirmed。**利空按剧烈程度评 ★4，不硬塞利好顶格。**
- **利空传闻+万亿巨头→打折不上手机（★★★ down, confirmed=false）**：「据报道 OpenAI/谷歌向五角大楼黑名单中资企业新加坡子公司供 AI」→ intensity=3, event_types=[], direction=down, confirmed=false, ticker_hint=["GOOGL","MSFT"], sector=["AI"]。理由：①`reportedly` 二手非官方 → confirmed=false；②GOOGL/MSFT 万亿巨头，单条合规传闻难成板块级异动 → 强度压到 ★3；③利空按自身剧烈程度评级。**★3 → 只上 TG 静音，不上手机（哪怕 GOOGL 在关注股内，因传闻+★3 不升级警笛）。**
- **领袖预言板块级重定价（★★★★★）**：「黄仁勋称Marvell将成下一家万亿公司」→ intensity=5, event_types=[3], direction=up, ticker_hint=["MRVL"], 联动=["COHR","LITE","GLW"]（光链溢出）, sector=["SMH","SOXX"]。理由：一句话即引发板块级重定价。
- **领袖温和背书 ≠ 满级（★★，不推）**：「黄仁勋在Adobe投资者会背书其AI创作系统」→ intensity=2, event_types=[3], is_push=false。理由：影响力虽跨到应用软件层，但只是**泛泛肯定、无极端目标价/无"下一个万亿"级表述、无具体订单**→力度温和。**关键区分：领袖发言要看"终局预言/明确荐股(★4-5)" vs "顺带称赞/温和背书(★2-3)"，不可一律打满。**
- **政府拨款但无上市标的→降级较大（★★，不推）**：「能源部拨款最多3.5亿支持四座煤电厂」→ intensity=2, is_push=false。理由：政府拨款本是利好，但金额小+无上市纯玩家可承接→财富效应无处落地，降级到 ★★（不因无标的归零，仍认利好底色，但降幅大到不推）。
- **纯A股事件不推**：「黄仁勋称机器人是韩国下一领域，A股机器人概念集体涨停」→ is_event=false, filter_reason="仅A股概念异动，无美股标的或直接产业链映射"。理由：涨的是A股，美股无直接对标标的→不推。
- **🔑 时效性降级：旧闻新报 ≠ 新催化剂（is_event=false，不推）**：「WSJ深度报道揭秘：白宫如何用关税筹码施压苹果、英伟达转向英特尔代工，政府90亿持股10%」→ is_event=false, notable=true, timeliness="retrospective", ticker_hint=["INTC","AAPL","NVDA"], filter_reason="旧闻新报：政府入股11个月前(2025.8)已发生并广泛报道，关税谈判一年前，苹果/英伟达合作此前已公开。WSJ新价值在因果链细节，不是新市场催化剂——核心事件早已price in。" 理由：**时效性是第一权重**。政府入股是真催化剂（类型1），但它发生在11个月前，不是今天。WSJ今天报道 ≠ 事件今天发生。即使权威媒体发表，timeliness=retrospective 的事件强度上限★2（实际判 is_event=false 因为所有核心事件均非新发生）。对比：「商务部**今天宣布**将补贴转为Intel股权」→ timeliness=immediate, ★5。**同一个催化剂，发生时间不同 → 评级天差地别。**
