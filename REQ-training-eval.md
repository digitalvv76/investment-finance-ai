## 0. 一句话定位（先纠正一个说法）

这**不是**"训练一个模型"那么简单，本质是三件事的组合：
**建带真实结果的标注数据集 → 用它校准/增强现有 LLM 评估器 → 用留出集客观验收。**
所以本项目的产物是「**数据集 + 校准方法 + 评估基准**」，不预设要 fine-tune。

## 1. 背景（为什么值得上流程）

评估器是全系统的"脑子"：判新闻的催化剂类型/强度/推不推。本周三个事故——美光**过推**、哨兵**零推(漏)**、deep_lane**编数据**——根子都是它判得不准或没被客观卡过。给它建可度量的训练+验收闭环，是对症。原型(`autolabel_training.py`)已证明"用真实行情自动标注"机制可行，并挖出命门=ticker 清洗。

## 2. 目标 / 非目标

**目标(Goals)**
- G1 评估器对催化剂**强度**与**推送决策**的判定，在留出测试集上达到可度量的准确率(见 §5)。
- G2 修正两类历史错误：过推(旧催化/宽补贴误判高强度) + 漏推(真异动被丢)。
- G3 深度分析**受益股+联动板块**深挖能力可评测（对应"广度不降级"原则）。
- G4 数据集能**自增长**（系统每天出题、市场给答案）。

**非目标(Non-goals)**
- 不建通用 MLOps/训练平台；不做实盘执行；不改推送渠道格式；本期不追求 fine-tune(见【待拍板1】)。

## 4. 需求（功能，每条可测）

- **R1 标注数据集**：每条样本含 `新闻 + ticker + 时间 + 真实反应(涨跌%) + 强度标签 + 方向 + is_push`。
- **R2 ticker 清洗**【命门】：标注前 ticker 必须清洗/核实，准确率达 §5 门槛，否则标签作废。
- **R3 标注方法**：①自动(真实行情回标，`autolabel_training.py`扩展) + ②人工锚点(稀有类如领袖预言/空头报告，手工，见现有 26 条种子)。
- **R4 覆盖平衡**：正面5类 + 负面5类(N1-N5) + **噪音真负例**(体育/政治/旧闻→不推) + 边界难例(宽补贴vs点状入股、新鲜vs隔夜)。**噪音反例是当前最缺的。**
- **R5 评估基准**：一个**人工核对过的留出测试集**（不参与任何校准/少样本），评估器在其上跑出 §5 指标。**盲测**：测试时只喂事件、**藏起真实答案**；且**任何已被用作 few-shot 样本的条目必须排除出测试集**，防泄露→假高分（见记忆 holdout-blind-eval）。选了路线 A(少样本)后这条是硬约束。
- **R6 反馈闭环**：新数据能定期自动并入（自增长），且新并入不污染留出集。
- **R7 防编数纪律**：任何环节缺真实行情，标签/分析**不得编造数字**（承接 deep_lane 硬门禁）。派生/无行情样本必须显式标注。

## 5. 验收标准（前置·可度量·整个项目朝这走）

> 数值为 **【建议·待 D4 确认】**。核心原则：**没有数字，不算验收。**

**A1 数据质量门**
- ticker 准确率 ≥ **95%**（抽样人工核 100 条）。
- 数据集 ≥ **300** 条，覆盖全部 5+5 类型 + ≥ 20% 噪音真负例。

**A2 评估器准确率（在留出测试集上，盲测）**
- 前提：测试集与 few-shot 样本**零重叠**（防泄露，见 R5 / 记忆 holdout-blind-eval）。测试只喂事件、藏答案。
- 强度判定：与 ground-truth 差 **≤1 星** 的比例 ≥ **80%**。
- 方向(涨/跌)命中 ≥ **90%**。
- 推送决策：is_push 的 **precision ≥ 75%**（控过推）且 **recall ≥ 70%**（控漏推）。

**A3 回归锚点（具名必过）**
- 美光$250B(隔夜旧催化) → **不响手机**。
- 哨兵曾漏的真异动(TSLA UBS上调/MRVL surge) → **响**。
- 深度分析无行情场景 → **不输出具体价格/建议**。

**A4 不劣化**
- 上线前**影子容器**对比现状：过推率↓、漏推率不升；A2 指标不低于现状基线。

## 6. 风险与假设（必须正视，否则文档是走过场）

- **RK1 相关≠因果**：股票在新闻后动，未必是这条新闻causing的（可能宏观/别的催化剂）。→ 缓解：优先取"单催化剂日"、多源确认；高风险类别用人工锚点兜底；自动标签定位为**噪音标签**，不当绝对真理。
- **RK2 日级近似**：原型用日涨跌，未做盘中对齐。→ D2 引入 T+4h 补充。
- **RK3 ticker 天花板**：清洗质量决定一切(命门)。达不到 A1 就**停工先修 R2**。
- **RK4 过拟合历史**：留出集必须严格隔离、且含未来新样本滚动验证。
