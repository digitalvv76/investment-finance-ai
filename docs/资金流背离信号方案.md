# 资金流背离信号方案

> 供投资专家评估。当前系统已接入东方财富资金流数据（22只美股，每日自动采集），需评估信号逻辑是否合理。

---

## 一、当前逻辑（待替换）

只看资金流方向，不看股价：

```
连续3天超大单同向 + 主力占比>15% → 推手机
连续3天超大单同向 + 主力占比8-15% → 推TG
```

**问题：** 股价涨 + 主力流入 = "量价齐升"，是正常追涨不是信号。当前逻辑会误推。

---

## 二、提议逻辑：价量背离框架

不看单日涨跌，看"涨跌"与"特大单流向"是否一致。背离 = 主力在逆散户方向操作。

### 底背离（买入信号）

> 股价大跌 + 特大单逆势净流入 = 主力挖坑吸筹，短期反弹概率大

| 条件 | 阈值 |
|------|------|
| 股价 | 3日累计跌幅 ≥ 10% |
| 特大单 | 3日累计净流入（或微幅流出 ≤ 市值 0.1%） |
| 主力占比 | 绝对值 ≥ 12%（打折后，对应东财数据虚高） |

**信号强度：**
- ★★★ 强底背离：跌幅 ≥ 15% + 特大单大幅净流入 + 占比 ≥ 18%
- ★★ 标准底背离：跌幅 ≥ 10% + 特大单净流入 + 占比 ≥ 12%
- ★ 弱底背离：跌幅 ≥ 10% + 特大单微幅流出 + 占比 < 12%

### 顶背离（卖出/减仓信号）

> 股价大涨 + 特大单逆势净流出 = 主力拉高出货，短期回撤概率大

| 条件 | 阈值 |
|------|------|
| 股价 | 3日累计涨幅 ≥ 10% |
| 特大单 | 3日累计净流出（或微幅流入 ≤ 市值 0.1%） |
| 主力占比 | 绝对值 ≥ 12% |

**信号强度：**
- ★★★ 强顶背离：涨幅 ≥ 15% + 特大单大幅净流出 + 占比 ≥ 18%
- ★★ 标准顶背离：涨幅 ≥ 10% + 特大单净流出 + 占比 ≥ 12%
- ★ 弱顶背离：涨幅 ≥ 10% + 特大单微幅流入 + 占比 < 12%

### 不产生信号的情况

| 场景 | 说明 |
|------|------|
| 股价涨 + 主力流入 | 量价齐升，正常追涨，不推 |
| 股价跌 + 主力流出 | 量价齐跌，正常抛售，不推 |
| 股价波动 < 10% | 幅度不够，不值得关注 |
| 主力占比 < 8% | 机构参与度太低，噪音 |

| 信号强度 | 通道 | 示例文案 |

## 六、待评估问题

1. 阈值是否合理？（跌幅 10%、主力占比 12%）需要根据实际数据回测校准
2. 是否区分持股/关注股？（持仓股的背离信号权重是否应该更高）
3. 顶背离是否需要更谨慎？（卖出信号比买入信号更容易引起用户操作，是否需要更高门槛）
4. 3日窗口是否合适？还是应该也看 5 日/10 日？
