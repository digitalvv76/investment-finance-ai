# 资金流背离信号模型 — 生产级 V2.5 终版

> 版本: 2026-07-16 · 数据源: Futu OpenD · 覆盖: 美股 + 港股
> **状态: ✅ 生产就绪** · 法则: A→F 六条 · 核心锚点: 特大单 (Super Big)

---

## 🚦 核心执行摘要

> **不看单日涨跌，看涨跌与特大单流向是否"对着干"。**

| 信号方向 | 价格 | 特大单（唯一锚点） | 实战结论 |
|:---:|:---|:---|:---|
| 🔴 顶背离 | 涨 | 净流出 | 准备减仓 |
| 🟢 底背离 | 跌 | 净流入 | 准备建仓 |
| ⚪ 无背离 | — | 同向 | 不推送 |

### 处理层级（严格先后顺序）

```
第1步（必做）→ 扫描背离 → 🔴顶 / 🟢底 / ❌无
  ↓ 无背离 → 终止，不推送
  ↓ 有背离 → 第2步
第2步（辅助）→ 六法则(A-F)校验 → 对背离做置信度升/降级
  ↓
最终输出 → ★★★ STRONG / ★★ STANDARD / ★ WEAK
```

### ⚠️ 开发铁律（P0级）

资金分类的金额切割标准**完全信任并跟随富途底层数据源**。策略代码**严禁**出现针对 `super_big_net` 或 `big_net` 的金额硬编码阈值。我们只做**字段读取、合并计算与逻辑运算**。

## 一、核心信念

**不看单日涨跌，看涨跌与特大单流向是否一致。**
一致 = 趋势延续；背离 = 拐点临近。

一切技术指标都可以造假，但大资金的真金白银不会说谎。特大单（单笔 ≥ 100 万元）无法被机构隐藏——它是整个分析体系的锚点。

---

## 二、订单分类体系

| 层级 | 字段名 | 来源 | 定义 | 市场角色 |
|:---|:---|:---|:---|:---|
| **★ 特大单** | `super_big_net` | Futu `super_in_flow` | 单笔 ≥ 100 万元 | **机构/主力** — 趋势发动者，锚点 |
| 大单 | `big_net` | Futu `big_in_flow` | 单笔 20-100 万元 | 游资/大型私募 — 趋势加速器 |
| **主力（合计）** | `main_net` | **计算**: `super + big` | 特大单+大单 | 防机构拆单规避检测 |
| 中单 | `mid_net` | Futu `mid_in_flow` | 单笔 4-20 万元 | 中大户 — 情绪缓冲垫 |
| 小单 | `small_net` | Futu `sml_in_flow` | 单笔 < 4 万元 | 散户 — 反向指标 |

### 为什么主力 = 特大单 + 大单？

机构可以通过将一笔 500 万的订单拆成 25 笔 20 万的单子来规避"特大单"检测。将特大单和大单合并统计，能堵住这个漏洞（V2.1 P0）。

### 价格数据

股价和涨跌幅由 Futu `request_history_kline`（日K线）直接提供，不再依赖 yfinance。数据源统一。

## 四、背离信号定义

### 底背离 — 🟢 重点关注

> 股价暴跌 + 特大单逆势净流入（或仅微幅流出）→ 主力在吸筹

| 股价表现 | 特大单表现 | 信号强度 | 结论 |
|:---|:---|:---:|:---|
| 跌 > 10%（单日或 3 日累计） | 净流入 > 0 | ★★★ | 强底背离，短期反弹概率极大 |
| 跌 > 5% | 净流入 > 0 | ★★ | 底背离，关注 |
| 跌 > 10% | 净流出 < 总流出 10%（几乎没跑） | ★ | 弱底背离，主力没跟着砸 |

### 顶背离 — 🔴 风险预警

> 股价大涨 + 特大单逆势净流出（或仅微幅流入）→ 主力在出货

| 股价表现 | 特大单表现 | 信号强度 | 结论 |
|:---|:---|:---:|:---|
| 涨 > 10%（单日或 3 日累计） | 净流出 > 0 | ★★★ | 强顶背离，短期回撤概率极大 |
| 涨 > 5% | 净流出 > 0 | ★★ | 顶背离，警惕 |
| 涨 > 10% | 净流入 < 总流入 10%（几乎没跟） | ★ | 弱顶背离，主力没参与狂欢 |

### 不产生信号

| 场景 | 原因 |
|:---|:---|
| 股价涨 + 主力流入 | 量价齐升，正常追涨 |
| 股价跌 + 主力流出 | 量价齐跌，正常抛售 |
| 股价波动 < 5% | 幅度不够 |
| 主力占比 < 8% | 机构参与度太低，噪音 |

## 五、辅助确认法则（六条）

### 法则 A：小单反向确认

| 组合 | 效果 |
|:---|:---|
| 底背离 + 小单净流出（散户恐慌抛售） | ⬆️ 信号增强 — 底部确认 |
| 顶背离 + 小单净流入（散户疯狂追涨） | ⬆️ 信号增强 — 顶部确认 |
| 背离 + 小单同向（散户跟机构走） | ⬇️ 信号降级 — 可能是真趋势 |

### 法则 B：连续性确认

| 特大单 3 日方向 | 股价 | 含义 |
|:---|:---|:---|
| 连续净流入 | 横盘或微跌 | 主力在持续建仓 |
| 连续净流出 | 横盘或微涨 | 主力在持续减仓 |
| 进出交替 | — | 背离不成立，主力在做波段 |

### 法则 C：主力占比定强度

主力占比 = (特大单+大单) / |总成交额| × 100%

| 占比 | 调整 |
|:---|:---|
| < 1% | 机构参与度低，信号 **降一级** |
| 1-5% | 正常参与，维持 |
| 5-10% | 机构主导，信号 **升一级** |
| > 10% | 极端信号，警惕数据异常 |

### 法则 D：合力检测（新增）

健康走势需要资金共振，危险走势来源于结构失衡。

| 3日资金结构 | 含义 | 调整 |
|:---|:---|:---|
| 特大/大/中/小 **全部净流入** + 股价涨 | 四维共振，健康上涨 | 信号**升一级** |
| 特大/大/中/小 **全部净流出** + 股价跌 | 一致抛售，趋势确认 | 维持 |
| 仅中单/小单净流入，特大单持平或流出 + 股价涨 | **散户独舞**，主力未参与 | 信号**降两级** |
| 仅中单/小单净流出，特大单持平或流入 + 股价跌 | 散户恐慌，主力未跟 | 可能为洗盘尾声 |

### 法则 E：散户陷阱（新增）

> 股价大涨（3日累计 > 5%）+ 特大单持平或流出 + 小单疯狂流入（超过特大单绝对值 3 倍以上）= **诱多陷阱**

训练数据验证：
- 6/15：+19.6% 但特大单 -4690万，小单 +6.12亿 → 随后连续暴跌
- 7/6：+7.26% 但特大单仅 -33万，小单 +2.46亿 → 次日暴跌 -15%

触发时信号**直接降两级**，标题标注「⚠️ 散户陷阱」。

### 法则 F：黄金坑（新增）

> 股价暴跌（3日累计 < -10%）+ 资金流出极小 + 特大单逆势净流入 = **黄金坑**

逻辑：巨大的跌幅没有对应的巨大抛盘，说明套牢盘已经躺平。此时少量机构资金就能托住底。

训练数据验证：
- 7/7：跌 -15% 但特大单 +5710万，总流出极小 → 次日反弹 +7.26%
- 6/22：跌 -16% 但特大单仅流出 -3809万（相对其他单极小）→ 次日止跌

触发时信号**升一级**，标题标注「★ 黄金坑」。

## 六、分析流程

```
第〇步：排除噪音日
  ├── 财报日 ±1天 · FOMC/CPI/非农 · 期权到期 · VIX>35
  └── → 降权但不作废

第一步：趋势定位
  └── MA7/MA25/MA99 · 52周高低点 · 均线乖离率

第二步：搜寻背离
  └── 逐日扫描 涨幅 vs 特大单方向

第三步：辅助确认（六法则）
  └── 小单反向(A) · 连续性(B) · 主力占比(C) · 合力(D) · 陷阱(E) · 黄金坑(F)

第四步：情景推演
  └── 三种情景(乐观/中性/悲观) + 概率 + 具体验证条件
```

## 八、代码层信号计算

### `compute_divergence_signal()` 算法

```
输入: List[FundFlowDay] (≥3日)
输出: {signal, strength, detail, details}

1. 取最近 3 日
2. 逐日比较:
   - price_delta = day[i].change_pct - day[i-1].change_pct
   - flow_delta = day[i].super_big_net - day[i-1].super_big_net  ← 特大单锚点
3. 背离判定:
   - price_delta>0 && flow_delta<0 → 顶背离计数+1
   - price_delta<0 && flow_delta>0 → 底背离计数+1
4. 六法则检测:
   - 合力: all 4 tiers same direction?
   - 散户陷阱: cum_price>5 && super≤0 && small>|super|*3
   - 黄金坑: cum_price<-10 && super>0 && total_outflow<|super|*5
5. 信号输出:
   - ≥2日背离 + 黄金坑 → bullish_divergence +20 strength
   - ≥2日背离 + 散户陷阱 → bearish_divergence +20 strength
   - ≥2日背离 → bullish/bearish_divergence
   - 1日背离 → warning
   - 四维共振 → confirmation (strength 75)
   - 散户独舞 → warning (strength 40)
   - 其他 → none

details:
  - cum_main_3d: 主力(super+big) 3日累计
  - cum_super_big_3d: 特大单 3日累计
  - latest_main_pct: 最新日主力占比
  - continuity: continuous_inflow | continuous_outflow | mixed
  - participation: extreme(>15%) | strong(>8%) | normal(>3%) | low
  - all_tiers_aligned: bool (四维共振)
  - retail_trap: bool (散户陷阱)
  - golden_pit: bool (黄金坑)
  - modifiers: list of active law tags
```

## 十、特殊标的处理

### 杠杆 ETF（SOXL、TQQQ、SQQQ 等）
- 背离阈值提高到 ±15%（而非 ±10%）
- 优先用底层指数（SOXX、QQQ）资金流交叉验证
- 小单占比 > 80% 为额外散户情绪信号

### 小盘股 / 低流动性
- 绝对金额门槛降低（日成交 < $10M 时机构进出受限）
- 重点看主力占比而非绝对金额
- 信号置信度自动降一级

### 港股
- 港股通资金流作为北水补充确认
- 波动更大，背离阈值放宽到 ±15%
